Ошибка MSVCP / VCRUNTIME?

Если программа не запускается, удалите старые версии Visual C++ и установите пакет:

Скачать Visual C++ Runtimes

Распакуйте и запустите install_bat.all от имени администратора.

Как обновить подписки?

Для NekoRay / Throne: Нажмите кнопку «Настройки» → «Группы» → «Обновить все подписки».

Скачать V2Box
  1. Скопируйте конфиг.
  2. В приложении перейдите во вкладку Config.
  3. Нажмите «+» (справа сверху) → Добавить подписку.
  4. Введите любое название, а в поле URL вставьте ссылку.
  5. Дождитесь проверки и нажмите на название конфига для выбора.
  6. Нажмите переключатель внизу для подключения.

Как обновить конфиги?

Перейдите во вкладку «Config» и нажмите на иконку обновления слева от названия вашей подписки.

Скачать Hiddify
  1. Скопируйте конфиг.
  2. Нажмите Новый профиль → Добавить из буфера обмена.
  3. Важно: Перейдите в настройки и измените "Вариант маршрутизации" на "Индонезия".
  4. В настройках выберите "VPN сервис".
  5. Нажмите большую кнопку подключения.
  6. Для сменя сервера (при включенном VPN) перейдите во вкладку «Прокси».
